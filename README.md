# Proyecto 19 web_project_api_full

En este proyecto se completa el trabajo en el Back-End con:

- la creación y autorización de usuarios en el servidor (Google Cloud),
- el manejo de errores centralizados y
- el despliegue del proyecto

## Características del proyecto

- MongoDB
- Express
- Node.js
- Nodemon (Dev)
- ESLint (Dev)
